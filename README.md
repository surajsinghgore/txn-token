
# Txn-Token

This was developed to provide unique numbers of 16 digits


## Install



```bash
  $ npm install tnx-token
```
    
## Usage

Require this package
```js
const GenTxnToken=require('txn-token');
```
generate Unique tokens
```js
const token=GenTxnToken();
```
